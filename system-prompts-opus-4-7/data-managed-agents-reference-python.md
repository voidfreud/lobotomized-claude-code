# Managed Agents — Python

Use the official Anthropic Python SDK for Managed Agents when documented. The stable mental model is: create/update Agent and Environment resources, create a Session for each run, stream session events, and send messages/tool results back through session APIs.

Do not infer Python method names from cURL or TypeScript. Fetch the current Python SDK docs or source for exact namespaces, parameters, and event classes before writing code.
