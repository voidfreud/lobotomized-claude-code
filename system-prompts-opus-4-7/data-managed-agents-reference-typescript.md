# Managed Agents — TypeScript

Use the official Anthropic TypeScript SDK for Managed Agents when documented. The stable mental model is: create/update Agent and Environment resources, create a Session for each run, stream events, and send messages/tool results through session APIs.

Do not infer TypeScript method names from cURL or Python. Fetch the current SDK docs or source for exact namespaces, parameters, event types, and stream helpers before writing code.
