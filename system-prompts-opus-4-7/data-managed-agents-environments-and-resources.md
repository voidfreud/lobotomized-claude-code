# Managed Agents — environments and resources

Environments are reusable runtime templates for sessions. Resources are mounted into a session at creation time: files, repositories, memory stores, vault credentials, and related inputs.

Keep credentials host-side or in vault-backed resources. When the user asks for exact mount fields, environment config, or resource lifecycle, fetch the current docs before writing code.
