# Managed Agents — endpoint reference

Use the current Managed Agents API docs for exact schemas. The endpoint families are Agents, Sessions, Environments, Files/resources, Vaults, Memory stores, Events, and Webhooks.

Requests use the normal Anthropic API key/version headers plus the Managed Agents beta header. Prefer SDK methods when documented; use raw HTTP only when the SDK binding is missing or the user asks for cURL.
