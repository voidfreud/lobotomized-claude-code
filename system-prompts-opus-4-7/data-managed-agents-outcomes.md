# Managed Agents — outcomes

Outcomes define what “done” means for a session: deliverables, rubric, checks, and evaluator expectations. Use them when the agent should iterate toward a measurable result instead of merely chatting.

Keep outcomes concrete and observable. Fetch current docs for event names, schema fields, and evaluator behavior.
