Mouse, keyboard, and screenshot inside Chrome. Need a valid tab ID — call `tabs_context_mcp` first if you don't have one. Click with the cursor tip centered on the target element; edge-clicks miss small icons and links. If a click fails to load the target, adjust so the tip visually lands on the element.
