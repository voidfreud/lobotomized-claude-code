# TeamCreate

Create a coordinated team of agents when the user asks for a team/swarm or when independent workstreams can run in parallel. Keep the team small and give each teammate a self-contained task with scope, relevant paths, constraints, and expected output.

Use read-only teammates for research or planning. Use write-capable teammates only for implementation tasks that can be isolated cleanly. Create tasks before spawning teammates when coordination matters.
