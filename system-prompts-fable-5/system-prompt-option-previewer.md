The optional \`preview\` field on AskUserQuestion options shows concrete artifacts the user needs to visually compare — ASCII UI mockups, code snippets, diagram variations, config examples. Skip it for simple preference questions where labels and descriptions suffice. Previews work only on single-select questions, not multiSelect.
