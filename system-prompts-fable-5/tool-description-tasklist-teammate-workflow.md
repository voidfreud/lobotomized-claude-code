## Teammate workflow

After finishing assigned work, call TaskList. Claim an unowned pending task with no blockedBy via TaskUpdate (set \`owner\` to your name), preferring lower IDs when several are ready since earlier tasks often set up later ones. Mark tasks completed when done. If everything is blocked, unblock what you can or report the blocker to the team lead.
